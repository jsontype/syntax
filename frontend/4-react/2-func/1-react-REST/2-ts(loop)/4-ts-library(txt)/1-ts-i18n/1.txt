[TS 기반 리액트 앱에 i18n 도입하기]

1. 설치
  $ npm i i18n react-i18next i18next --save

1. tsconfig.json 수정 : 이 부분이 없다면, 추가해야 i18n 임포트 사용이 가능함
  "compilerOptions": {
    // ...
    "include": [
      "src",
      "@types"
    ]
  },

1. src/i18n/index.js 생성 : i18n 기본 설정하기

1. src/i18n/locales/[en, ko, ja]/index.js 생성 : 언어에 적용할 json 그룹 객체 선언해둔다.

1. src/i18n/locales/[en, ko, ja]/[counter.json, todos.json, movies.json] 생성 : json 파일을 만들때는, 아래와 같이 쉼표가 끝 부분에 있으면 에러가 난다.
  {
    "a": "a",
    "b": "b",
    "c": "c", // , 를 끝에 넣으면 에러가 난다. VSCode 상에서 빨간줄 에러를 안 띄울 수도 있으니 주의
  }

1. src/index.js 수정 : i18n 폴더안의 기본설정 파일을 도입해서 사용할 수 있도록 세팅

1. src/components/[counter, todos, movies]/index.js 수정 : i18n을 직접 사용하는 화면 작성

1. src/App.js 수정 : 상단 메뉴에 영/한/일 버튼 만들고, index로부터 받은 i18n 함수를 이용해서 언어 변경

1. 테스트
  $ cd app1
  $ npm start
