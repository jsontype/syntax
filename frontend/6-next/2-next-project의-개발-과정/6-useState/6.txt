1. use client
  Next 앱에서 useState를 쓰려면 'use client' 명령어가 필요하다.
  넥스트 앱에서는 컴포넌트를 만들면 기본적으로 Server 컴포넌트가 되기 때문이다.

1. Server 컴포넌트 vs Client 컴포넌트
  - Server 컴포넌트
    Server 컴포넌트는 백엔드 부분으로, DB나 파일 시스템에 기반한 작업들을 주로 다룬다.
    Server 컴포넌트들은 기본적으로 다음과 같은 제약이 있다.
      useState를 사용하지 않는다.
      useEffect를 사용하지 않는다.
      props에 함수를 전달할 수 없다.
      onClick 같은 이벤트를 포함하지 않는다.
  - Client 컴포넌트
    넥스트 앱에서는 파일 맨 위에 'use client' 넣고 만들면 Client 컴포넌트가 된다.
    넥스트가 아닌 리액트 앱에서는 'use client'가 없어도 기본적으로 Client 컴포넌트이다.
