1. Next15 프로젝트 시작하기
  $ npx create-next-app@15.2.4 app3 # 2025년 3월 기준 latest 버전
    Typescript? > Yes
    ESLint? > Yes
    Tailwind CSS? > Yes (No로 하면 자동으로 css module로 설정됨. 즉 page.module.css이 자동 생성됨)
    src/? > Yes
    App Router? > Yes
    Turbopack? > No
    import alias? > No (No로 하면 /src 절대경로가 기본값인 @이 됨)
  $ cd app3
  $ npm i
