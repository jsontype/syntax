1. 뷰3 스토어 : Vue 3의 Composition API에서 computed를 사용하여 get과 set 메서드를 정의하는 것은 일반적인 접근 방식입니다. 이미 정의된 uploadedSubmitDocumentFiles의 set 메서드를 사용하여 파일 목록을 업데이트하려면, set 메서드를 직접 호출하는 것이 아니라, 해당 computed property에 값을 할당함으로써 set 메서드를 간접적으로 호출할 수 있습니다. ---> 이부분 아래코드 같은 거 자세히 설명좀 해줘. 
// Computed
const count = computed({
  get: (): FileItem[] => {
    return countStore.count
  },
  set: (files: FileItem[]): void => {
    countStore.count = count
},
})
이런식으로 스토어에 값을 이용해 카운터를 구현하려고 하는 중임.
