[Storybook]

1. 사전준비 : init 앱을 가지고 온다.

1. 사전준비 : src/components/HelloWorld.vue 삭제

1. 라이브러리 설치
  $ npm i -D @storybook/vue@6
  $ npx sb init
  위 명령어로 ".storybook/main.ts"이 생성되었는지 확인. 설정이 필요하면 해당 파일의 내용을 변경하면 된다.

1. src/components/Button/ButtonEx.vue 생성
  ```
    <template>
      <button @click="onClick">{{ count }}</button>
    </template>

    <script setup>
    import { ref } from 'vue';

    const count = ref(0);

    const onClick = () => {
      count.value++;
    };
    </script>
  ```

1. src/components/Button/ButtonEx.stories.js 생성
  ```
    import { defineComponent } from "vue";
    import ButtonEx from "./ButtonEx.vue";
    
    export default {
      title: "ButtonEx",
      component: ButtonEx,
    };
    
    const Template = (args) =>
      defineComponent({
        components: { ButtonEx },
        setup() {
          return () => <ButtonEx {...args} />;
        },
      });
    
    export const Primary = Template.bind({});
    Primary.args = {
      count: 0,
    };
  ```

1. src/App.vue 수정 : 컴포넌트를 호출한다.
  ```
    <template>
      <ButtonEx />
    </template>

    <script setup lang="ts">
    import ButtonEx from '@/components/Button/ButtonEx.vue'
    </script>

    <style scoped>
    </style>
  ```

1. 테스트
  $ cd app1
  $ npm run dev
  $ npm run storybook // 안됨
