[Local Components]

1. 사전준비 : 프로젝트를 새로 깔지 않고, 이전 챕터에서의 코드를 그대로 카피해서 가지고 온다.

1. Components란?
  웹 애플리케이션의 UI를 구성하는 독립적이고 재사용 가능한 파츠.
  이를 통해 복잡한 UI를 작고 독립적인 단위로 나누어 관리할 수 있다.
  명명은 파스칼 케이스로 작성한다.

1. Local Components vs Global Components
  Local Components : 특정 컴포넌트에서만 사용하는 컴포넌트. 사용하기 위해서는 Import가 필요하다.
  Global Components : 여러 컴포넌트에서 공통으로 사용하는 컴포넌트. 특별히 Import가 필요없다.

1. main.ts 수정
  ```
    import { createApp } from 'vue'
    import App from './App.vue'

    createApp(App).mount('#app')
  ```
  
1. style.css 삭제

1. App.vue 수정
  ```
    <template>
      <HelloWorld />
    </template>

    <script setup lang="ts">
    import HelloWorld from './components/HelloWorld.vue'
    </script>

    <style scoped>
    </style>
  ```

1. components/HelloWorld.vue 수정 : 자식 컴포넌트에게 프롭을 전달. 태그로 불러올 때는 파스칼 케이스, 케밥 케이스 둘다 가능하다.
  ```
    <template>
      <HelloWorld />
      <hello-world />
    </template>

    <script setup lang="ts">
    import HelloWorld from './components/HelloWorld.vue'
    </script>

    <style scoped>
    </style>
  ```

1. 테스트  
  $ cd app1
  $ npm install
  $ npm run dev
