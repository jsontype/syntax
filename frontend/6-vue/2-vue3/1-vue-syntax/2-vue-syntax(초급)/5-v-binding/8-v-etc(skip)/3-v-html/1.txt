1. ㅇㅇ
  Raw HTML: v-html 디렉티브를 사용하면 HTML 코드를 직접 렌더링할 수 있다.
  예를 들어, <div v-html="htmlContent"></div>와 같이 사용해서,
  htmlContent 데이터에 저장된 HTML 코드를 페이지에 직접 렌더링한다.
  이때, htmlContent 데이터에 저장된 HTML 코드는 브라우저에 의해 HTML 태그로 해석되어 렌더링된다.
  * 그러나 이렇게 HTML 코드를 직접 렌더링하는 것은 보안상 위험하므로(개발자도구에 그대로 보여짐) 내용을 확인하고 사용해야 한다.
