[Vue2 Apollo]

1. GraphQL이란?
  BFF를 구현하기 위해 구성된, "서버용 스키마 언어"와 "클라이언트용 쿼리 언어"를 말한다.
  GraphQL는 프론트엔드와 백엔드 간의 통신을 용이하게 하는 것을 목표로 한다.

1. Apollo란?
  클라이언트와 서버 앱에서 각각 GraphQL을 사용하는 데 도움이 되는 도구를 제공하는 라이브러리이다.
  Vue에서는 vue-apollo, React에서는 react-apollo 라이브러리를 제공한다.

1. 사전준비 : init 앱을 가지고 온다.
  이번 챕터에서는 vue2 init 앱으로 진행한다.

1. 설치
  $ cd app1
  $ npm install --save vue-apollo graphql apollo-boost
    (= yarn add vue-apollo graphql apollo-boost)
    (보다 세밀한 제어를 원한다면 apollo-boost 대신 apollo-client를 설치하면 된다.)
  VSCode 확장 설치 : Apollo GraphQL

1. apollo.config.js 생성
  module.exports = {
    client: {
      service: {
        name: "app1",
        // URL to the GraphQL API
        url: "http://localhost:3000/graphql",
      },
      // Files processed by the extension
      includes: ["src/**/*.vue", "src/**/*.js"],
    },
  }
