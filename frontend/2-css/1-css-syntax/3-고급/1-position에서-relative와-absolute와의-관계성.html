왜 부모 태그에 Relative를, 자식 태그에 Absolute를 사용하는가?

Relative Positioning: 이는 요소를 기본 위치에서 상대적으로 이동시키는 방법이다. 이것은 요소가 원래 있어야 할 곳에서 상대적인 위치를 차지하게 된다. 하지만, 이 요소의 원래 공간은 여전히 페이지 레이아웃에 영향을 미치게 된다.

Absolute Positioning: 이는 요소를 페이지의 흐름에서 완전히 분리시키고, 가장 가까운 상대적(position: relative) 또는 절대적(position: absolute/fixed) 위치를 지정한 조상 요소에 대해 상대적으로 배치한다. 만약, 이러한 조상이 없다면, 문서의 body를 기준으로 배치된다.

요약: 이러한 이유로,
부모 태그에 position: relative를 적용하고 자식 태그에 position: absolute를 적용하면, 자식 태그는 부모 태그 내에서 상대적 위치를 갖게 된다. 이는 레이아웃 설계 시 요소들의 정확한 배치를 더 쉽게 해준다.
