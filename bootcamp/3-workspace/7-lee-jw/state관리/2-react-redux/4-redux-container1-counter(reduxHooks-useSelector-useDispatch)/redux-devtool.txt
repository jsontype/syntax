[Redux 개발자도구 적용하기]

1. Redux 개발자 도구 기능
    state를 조회
    dispatch된 액션과 그에 따른 state 변화 이력을 트레이스
    액션을 직접 디스패치 할 수도 있다.

1. Redux 개발자 도구 설치
    크롬 웹스토어에서 Redux DevTools 확장 프로그램을 설치한다.
    $ cd app1
    $ npm i redux-devtools-extension (= yarn add redux-devtools-extension)
