object-fit = CSS의 object-fit 속성은 image나 video등의 미디어를, 컨테이너안에 어떻게 끼울지를 설정합니다.

object-fit: fill = 종횡비를 콘텐츠 박스 크기만큼 늘린다.

object-fit: contain = 종횡비를 유지한 채 콘텐츠 박스 크기만큼 확대 축소한다.

object-fit: cover = 종횡비를 유지한 채 콘텐츠 박스의 종횡 중 큰 쪽의 크기만큼 확대 축소한다.

object-fit: none = 종횡비를 수정하지 않는다.

object-fit: scale-down = contain과 같지만, 오브젝트의 실제 크기가 작다면 none처럼 종횡비를 수정하지 않는다.
