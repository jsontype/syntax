1. VSCode 확장 설치 필요
  Vue Language Features (Volar) : vue 개발용
  Vue VSCode Snippets : vbase라고 치면 vue 확장자에 한해서 자동 컴포넌트 생성



1. 컴포넌트 만드는 법 요약
  자식컴포넌트 만든다. → 자식컴포넌트 익스포트한다. → 부모컴포넌트에서 임포트한다. → 부모컴포넌트에서 사용한다.



1. src/App.vue 수정
  <template>
    <div class="hello">
      <Hello />
      <Hello />
      <Hello />
      {{ name }}!
    </div>
  </template>

  <script>
  // 1. 컴포넌트 임포트한다.
  import Hello from './Hello'

  // export default App : 익스포트를 하기 위한 것으로, 템플릿 문법에 의해 적용된다.
  export default {
    name: 'App',
    // 2. 쓰고자 하는 자식컴포넌트들을 components 객체 안에 담는다.
    components: {
      Hello, // = HelloWorld: HelloWorld,
    },
    data() {
      return {
        name: 'Yang',
      }
    }
  }
  </script>

  <style>
  .hello {
    font-size: 22px;
  }
  </style>




1. src/Hello.js 생성 : vbase 입력시 함수형 컴포넌트 자동 생성하므로 참조
  <template>
    <div class="text">
      Hello!
    </div>
  </template>

  <script>
  export default {
    name: 'Hello'
  }
  </script>

  <!-- 3. App.vue을 제외한 모든 자식컴포넌트들은 style에 scoped를 배정해서 중복을 방지한다. -->
  <style scoped>
  .text {
    color: gray;
  }
  </style>



1. 테스트
  $ cd app1
  $ npm start
