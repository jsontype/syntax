[vi, vim 필수 기본 명령어]

1. Vi, Vim 설치 : 맥은 불필요
    vim.org에서 튜토리얼을 확인하고 설치

1. Vi, Vim 열기 : 환경구축을 위해 config 파일 수정할 때 등에 사용한다.
    vi filename
    vim filename

1. 모드 전환
    "i, a, o" 중 하나 : 일반 -> 입력모드로 전환    
    "esc" : 입력 -> 일반모드로 전환    
    ":, /, ?" 중 하나 : 일반 -> 명령행모드로 전환

1. 명령행모드 명령어
    :w : 저장
    :q! : 저장없이 강제 종료 (그냥 :q만 하면 이후 에러 발생시키고, 작업하던 것이 temp로 남음)
    :wq : 저장 후 종료
    :w [파일명] : 다른이름으로 저장
    :w! [기존 파일명] : 기존 파일에 강제 덮어쓰기

1. 일반모드 명령어
    gg : 첫번째 행으로 이동
    shift v(비주얼 라인모드) + shift g(현재 위치 이하 모두 선택) : 모두 선택
    d : 삭제
    y : 복사
    p : 붙여넣기
    u : 되돌리기
