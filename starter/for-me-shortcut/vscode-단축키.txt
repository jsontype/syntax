ctrl(cmd) + b : 사이드바 닫기/열기 토클
ctrl(cmd) + , : 설정 화면 열기
ctrl(cmd) + p : 파일명 검색
ctrl(cmd) + shift + p : 설정 검색
    * 맥에서 가끔 한글 키보드 먹히는 현상 해결법 :
        ctrl(cmd) + shift + p 누른 후, 표시언어구성 누른 다음, ko 변경하고 재시작
        ko로 되어있더라도, 다시 설정하고 재시작을 해야 고쳐짐.
터미널 열고 code . : 해당 디렉토리 쉽게 열기
! (치고 Fn + Ctrl + Space) : HTML 자동생성

cmd + opt + 위아래키 : 같은 줄에 있는 곳에 동시 커서 띄우기
opt + 마우스클릭 : 원하는 곳에 동시 커서 띄우기
cmd + d : 동일한 변수명 동시커서 띄우기 (누를 때마다 하나씩 추가)
opt + 방향키 : 단어 단위로 이동
opt + shift + 방향키 : 단어 단위로 블록 선택
