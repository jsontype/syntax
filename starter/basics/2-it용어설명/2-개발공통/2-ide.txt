[IDE의 정의]
    IDE란 : 통합 개발 환경(Integrated Development Environment)의 약자로, 코딩을 할 수 있는 개발툴
    * 메모장도 일종의 개발툴이지만, 컴파일링, 디버깅을 포함하여 다양한 기능이 전무하므로 IDE라고는 하지 않는다.


[IDE의 종류]
1. vscode
    - 주로 사용하는 언어
        front : javascript
        back : php, python

    - 장점 : 터미널과 연동

    - 설치 : https://kr.vuejs.org/v2/guide/installation.html

    - 참고자료 : https://kr.vuejs.org/v2/guide/index.html

    - 추가 Extenstion 설치 목록 : 
        CSS Peek : html 안의 class="" 를 클릭하면 적용된 CSS로 바로 이동해줌
        Live Server : 코딩 실시간으로 브라우져에서 확인 가능, 아래 파란 옵션창에서 Go Live 클릭하면 나옴.

    - 기본 Settings 변경 목록 : 
        파일위치로 새 터미널 생성시키기 
            File – Preferences – Settings 를 열고, terminal.integrated.cwd 를 입력하여 찾아간 후, 대화상자에 다음 값을 넣는다.
            ${fileDirname}
