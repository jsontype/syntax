1. JSX 문법
    변수나 함수를 HTML 리턴안에 넣으려면 {}로 감싸면 된다.
    div class='' 대신, div className='' 을 사용한다.
    return 안에는 하나의 부모 태그만 가능 : div 쓰기 싫을 때는 fragment ( = <></> )

1. 테스트
    cd app1
    npm start
