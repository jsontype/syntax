<기본>
    전체 스크린샷 : cmd shift 3(커맨트 c 필요)
    부분 스크린샷 : cmd shift 4(커맨트 c 필요)
    잘라내기 : cmd c 한 다음, 붙여넣기할 때 cmd 옵션 v
    바탕화면 보기 : Fn F11
    스플릿뷰 : 화면 넓히는 초록 버튼에 마우스 대고 있다가, 화면 왼쪽/오른쪽으로 이동 버튼 나오면 누르기(안 뜨면 옵션키 눌러보기)
    맥북 바탕화면 이동 : F3키 누르고 원하는 화면 클릭
    특수문자, 기호 입력 : ctrl cmd space
    한자 입력 : 옵션 엔터 (기호는 안됨)
    앱 닫기 : cmd q
    창 닫기 : cmd w 
    켜진 앱 전체 보기 : 트랙패드 세손가락으로 쓸어올리기
    윈도우키 : 트랙패드 네손가락으로 오므리기 (단축키 없음) (네손가락으로 펴면 원래로 돌아옴)
    Del키 : Fn Delete
    스프레드시트 매크로용 shift ctrl alt 숫자키 : shift opt cmd 숫자키
    vs코드 확대/축소 : cmd shift - / cmd -
    크롬 확대 축소 : （일본어키보드에서) 시트하나 열고 cmd shift +, (영어키보드에서) cmd
    크롬 새로고침(F5) : cmd r
    시스템 종료(Alt F4) : ctrl 전원키
    백슬래쉬 : (영어키보드에서) opt ¥
    원화표시 : (한글키보드에서) shift @
    백틱표시 : (영어키보드에서) shift @ (한글키보드에서) shift opt @
    vs코드 한글 씹힐 때 : cmd shift p → 표시 언어 구성 → ko 선택 (이미 ko로 되어있더라도, "언어를 변경하는 행위"가 중요함)
    파인더에서 숨김파일 보기 : cmd shift .

<스펙타클(화면스플릿앱)>
    풀스크린 : cmd opt f
    센터 : cmd opt c
    3분의 1 : ctrl opt 방향키
    반절 이동 : cmd opt 방향키s
    위 반절 이동 : cmd ctrl 방향키
    아래 반절 이동 : cmd shift ctrl 방향키
    확대축소 : ctrl opt shift 방향키

<git 관리>
    terminal이나 vscode로 git을 관리한다. sourcetree는 조회랑 체크아웃만 하는 용도로 사용

<CMD>

<code .>
    * code . 로 열리지 않는 경우, cmd shift p 누른 뒤, code 입력, 첫번째 것을 인스톨하면 된다.
    * 만일 github 등으로 당겨온 앱이라면 npm i를 통해서 vue.cmd 웹빌드를 해야한다.
