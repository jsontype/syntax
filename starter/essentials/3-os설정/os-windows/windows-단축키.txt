<git 관리>
Source Tree 탭에서 git을 관리한다.

<CMD>
win + r : cmd 창 열기
shutdown -s -t [초] : 예약종료
shutdown -a : 예약취소
misinfo32 : 시스템정보

<Short Cut>
win+ctrl+d : 대쉬보드 추가
win+ctrl+f4 : 대쉬보드 삭제
win+ctrl+방향키 : 대쉬보드 전환
win+tab : 대쉬보드 작업보기
★★★ win+shift+s : 빠른 스크린샷

win+l : 락
win+x : 제어판메뉴
win+; : 이모티콘
win+g : 녹화시작
ctrl+alt+tab : 알트탭 고정
ctrl+shift+esc 작업관리자

win+p : 디스플레이 확장 복제 변경
