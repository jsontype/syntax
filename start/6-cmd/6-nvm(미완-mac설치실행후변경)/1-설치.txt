[nvm]

1. nvm이란?
    여러 개의 노드가 설치된 경우, 특정 노드 버전을 사용할 수 있게 해주는 CLI 툴
    현장 또는 앱마다 다른 노드 버전이 필요한 경우에 사용한다.



[Windows에서 설치]

1. 다음 주소에서 nvm-setup.exe 파일을 다운로드
    https://github.com/coreybutler/nvm-windows/releases

1. 해당 파일을 실행시키면 nvm이 설치가 된다.
    vscode를 재기동
    $ nvm -v


    
[Mac에서 설치]

1. brew를 활용하여 nvm 설치
    $ brew install nvm

    * 만약 설치도중 man 페이지 관련 에러가 발생한다면, 권한 설정을 해주자
        sudo chown -R $(whoami) /usr/local/share/man/man3 /usr/local/share/man/man5 /usr/local/share/man/man7
        Password:
        sudo chmod u+w /usr/local/share/man/man3 /usr/local/share/man/man5 /usr/local/share/man/man7

1. 환경 변수 설정
    - ~/.nvm 디렉토리 생성
        $ mkdir ~/.nvm
    - ~/.zshrc 설정 (bash를 사용한다면 ~/.bash_profile 설정)
        export NVM_DIR="$HOME/.nvm"
        [ -s "/usr/local/opt/nvm/nvm.sh" ] && . "/usr/local/opt/nvm/nvm.sh"  # This loads nvm
        [ -s "/usr/local/opt/nvm/etc/bash_completion" ] && . "/usr/local/opt/nvm/etc/bash_completion"  # This loads nvm bash_completion
    - 적용
        $ source ~/.zshrc
