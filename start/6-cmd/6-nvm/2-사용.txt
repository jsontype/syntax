[nvm 사용법] 

1. nvm에 설치된 node의 리스트를 확인하기
    $ nvm ls 또는 nvm list

1. 설치 가능한 LTS 찾기
    $ nvm list available

1. 특정 node 버전을 설치하기
    현재 설치할 수 있는 node 버전 리스트 확인 : https://nodejs.org/ko/download/releases/
    $ nvm install 18.16.0

1. nvm에서 특정 버전 node를 활성화 하기
    $ nvm use 18.16.0

1. nvm에서 특정 버전 node 삭제
    $ nvm uninstall 18.16.0

1. nvm안에 여러버전의 node중에 어떤것으로 사용할 지 default 버전 설정
    $ nvm alias default v18.16.0
    여러개의 node 버전이 설치되었을 경우 default 설정을 잘해줘야 편하다.
    디폴트 설정 후에 nvm ls 로 확인하면, 어떤 버전이 디폴트로 되어있는지 보여진다.
