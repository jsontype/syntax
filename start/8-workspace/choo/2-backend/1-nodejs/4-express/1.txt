1. 헷갈리지 않아야할 용어
  서버 : 통로 역할을 하는 것이고, 그래서 클라이언트 서버도 있고, 백엔드 서버도 있다.

2. 프론트엔드와 백엔드
  프론트엔드 : 사용자가 보는 화면
  백엔드 : 사용자가 보는 화면을 만들어주는 역할

3. 언어들
  백엔드 언어 : java (Spring, Springboot, Sprut, ....), c, c#, c++, python, nodejs, php, ruby, go, kotlin, scala, swift, rust, ...
  프론트엔드 언어 : javascript (React, Vue, Angular, Svelte, ...)
  데이터베이스 SQL : mysql, mariadb, oracle, mssql, mongodb, ...
