[리액트 네이티브 Event]
참조URL: https://reactnative.dev/docs/pressable

1. 사전준비 : 프로젝트를 새로 깔지 않고, 이전 챕터에서의 코드를 그대로 카피해서 가지고 온다.

1. Event
  onPress : 손가락을 눌렀다 뗌
  onPressIn : 손가락을 누른 시점
  onPressOut : 손가락을 뗀 시점
  onLongPress : 손가락을 길게 눌렀다 뗌 (500ms)
  delayLongPress : onLongPress의 기준을 변경
  onChange : input이 변경
  onChangeText : e.target.value 값을 e에다가 넣는다.
