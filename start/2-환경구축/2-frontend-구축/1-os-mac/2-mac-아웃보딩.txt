[맥 반납할 때 포맷]
참조URL: https://support.apple.com/ko-kr/HT212749 (포맷)

1. 맥북 포맷 : 시스템 설정 -> 일반 -> 전송 또는 재설정 -> 모든 콘텐츠 및 설정 지우기



[맥 반납할 때 삭제 : 포맷이 안될 경우]

1. 크롬에 연동된 개인 구글 계정 삭제
  회사 계정만 남겨두고, 그것 역시도 크롬 인터넷 기록/캐시는 삭제

1. 크롬 북마크 삭제
  회사 계정에서 북마크 삭제

1. 회사 지메일 계정 내의 drive 내의 공유아이템 모두 삭제
  특히, 플래너나 근무표, 또는 타프로젝트 관련된 것

1. 슬랙 개인귓말 삭제
  DM이나 개인적 귓말 삭제

1. 맥북 메모 삭제
  메모 전부 삭제
  
1. 개인 프로그램 삭제
  개인이 쓴 모든 애플리케이션 확인해가면서 삭제
  특히, AnyDesk등 원격 프로그램은 반드시 삭제  

1. 콘솔에서 any 검색후 로그 삭제
  맥북 cmd + space 누른 뒤 console 검색
  console 응용어플리케이션 선택
  모든 리포트 하나씩 찍어가면서 "any" 검색 (특히 로그 리포트 쪽)
  있다면 해당 로그를 삭제
    해당 로그에서 마우스 우클릭 후, "Finder에서 보기" 클릭
    console 응용어플리케이션을 끄고, 해당 *.log 파일 휴지통에 넣기
    휴지통 비우기
  다시 처음부터 반복하기 (any 로그가 없어질 때까지)

1. Apple ID 등록 해제 
  나의 찾기, 앱스토어, 시스템설정 좌측상단에 나의 Apple ID가 등록되어 있다면 해제하기  
  클라우드 로그아웃 : 애플아이콘 - 시스템 설정 - icloud 선택 - mac 찾기 - 오프 - 하단 사인아웃 선택 - 애플 아이디 넣는 인풋창 뜨면 작업완료

1. 연결기기 삭제
  블루투스 연결된 기기 해제 및 삭제
  와이파이 연결된 기기 해제 및 삭제



[맥 반납 및 현장 종료 후의 마무리 처리]

1. 모바일 연동 삭제
  모바일 iPhone Calendar 연동 삭제 : 설정 - 캘린더(iCloud 캘린더 아님) - 계정 - 현장의 이메일 클릭 - 계정 삭제 - 나의 iPhone에서 삭제
  모바일 Google Calendar 연동 삭제 : 앱 열기 - 우측 상단 유저 아이콘 - 기기 계정관리 - 해당 아이디 옆의 이 기기에서 삭제 버튼 클릭
  모바일 Chrome 브라우져 연동 삭제 : 앱 열기 - 우측상단 아이콘 - 최상단 본인아이디 클릭 - 계정 클릭 - 기기에서 삭제
  모바일 Authenticator 연동 삭제 :
    Google Authenticator의 경우 : 앱 열기 - 해당 계정 클릭 - 설정버튼 - 삭제
    MicroSoft Authenticator의 경우 : 앱 열기 - 해당 계정 클릭 - 편집 - 삭제

1. 개인PC 연동 삭제
  구글 크롬 이메일 계정 삭제
  깃허브 연동 삭제 : 우측 상단 유저아이콘 - 세팅 - Organizations - 해당 기관의 leave 버튼 클릭

1. 모바일 / 개인PC 공통 연동 삭제
  슬랙 계정 사인아웃 및 삭제

1. 슬랙에 연동된 워크스페이스 삭제
  데스크톱 앱에서 삭제하기 : 앱 열기 - 워크스페이스 general 채널에 들어가기 - 개인 프로필 아이콘을 클릭 - 우측에 열리는 개인 프로필 사이드바에서 햄버거 메뉴 클릭 - 계정 설정 클릭 - 웹 브라우저가 열림 - 계정 비활성화 클릭
  모바일 앱에서 삭제하기 : 데스크톱 삭제시 자동으로 사라짐
