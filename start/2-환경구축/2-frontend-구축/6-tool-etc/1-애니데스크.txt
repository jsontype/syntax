[애니데스크 설정]

1. 애니데스크란?
  다양한 이유로 개발중 타 PC에의 원격 조종이 필요한 경우 사용하는 툴이다.
  다양한 원격 조종 툴이 있지만, Full Access가 가능한 점, 편리하고 빠른 점에서 애니데스크를 가장 추천한다.

1. 애니데스크 설치
  "애니데스크"를 크롬에서 검색 후 설치

1. 애니디스크 설정
  우측 상단 햄버거 아이콘 클릭
  설정 클릭
  보안 클릭
  Permission Profile에서 Unattended Access에 비밀번호가 걸려있다면 해제하고 다른 사용자 허용 체크박스 해제
  Permission Profile에서 Full Access 선택후 다른 사용자 허용 체크박스 체크 후 비밀번호 설정
  사용하고자 하는 타 컴퓨터에서 해당 비밀번호를 저장으로 해서 비밀번호 입력후 등록해두기

1. 맥의 경우, 권한 부여하기
  버튼 -> 시스템 환경설정 -> 보안 및 개인정보보호 -> 개인정보보호 탭 -> 손쉬운 사용 -> AnyDesk 체크
  버튼 -> 시스템 환경설정 -> 보안 및 개인정보보호 -> 개인정보보호 탭 -> 화면기록 -> AnyDesk 체크
