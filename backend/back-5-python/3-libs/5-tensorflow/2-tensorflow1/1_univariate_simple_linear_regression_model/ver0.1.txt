1. flask, tensorflow 설치

> pip install --upgrade pip : pip3 최신 업그레이드
> python : v3.5 이상인지 확인
>>> exit() : 파이썬 화면으로부터 나감
> pip --version : v19.0 이상인지 확인
> pip install tensorflow

1. tensorflow 설치가 안될 경우의 절차
브라우저에서 https://www.tensorflow.org/install/pip?hl=ko#system-install 에 접속
브라우저에서 "시스템에 Python 개발 환경 설치"에 들어간 다음, 윈도우 탭의 Microsoft Visual C++ 다운로드
> pip install --user --upgrade tensorflow  # install in $HOME : 텐서플로우 라이브러리 설치
> python -c "import tensorflow as tf; print(tf.reduce_sum(tf.random.normal([1000, 1000])))" : python 반환되면 텐서플로우 성공 설치
