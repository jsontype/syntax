[웹소켓 다뤄보기]

1. 웹소켓이란?
    원래, 서버와 유저가 데이터를 주고받는 방법은 다음과 같다.
        1. 유저가 서버에게 http를 요청한다.
        2. 서버가 유저에게 답장으로 데이터를 보낸다.
    서버로부터는 절때 먼저 선톡을 하지 않는다. 따라서 매번 유저가 요청을 해야 데이터를 받는다.
    그러나 요청을 보내지 않아도 2초마다 코인 가격을 수신받고 싶다면?
    그 해결책이 WebSocket이다. 이는 양방향 통신을 가능하게 한다.

1. 웹소켓을 써서 양방향통신 서버를 구축해보기    
    npm i express
    npm i ws
    index.html 작성
    3.js 작성
    node 3.js
    브라우저에서 localhost:8080에 들어가 확인

1. 참조
    socket.io 라이브러를 쓰면 다음 기능들이 자동 포함되어 더 편하게 구축 가능 : 연결끊기면 자동재접속가능, 웹소켓 접속자마다 자동 id 부여, 모든 웹소켓 유저에게 전체메세지 전송가능, 웹소켓방을 생성가능
