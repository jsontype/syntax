[PR Agent]

1. PR Agent란?
  pr-agent(Pull Request Agent)는 GitHub Pull Request를 자동으로 분석하고 리뷰하는 AI 기반 도구이다.
  
  주요 기능:
  - 자동 PR 리뷰: 코드 변경사항을 분석하여 잠재적 문제점을 찾아줌
  - 코드 품질 검사: 버그, 보안 취약점, 성능 이슈 등을 감지
  - 문서화 제안: 코드에 대한 설명이나 주석을 자동으로 생성
  - 테스트 커버리지 분석: 테스트가 충분한지 확인하고 개선점을 제안
  - 리뷰 요약: PR의 주요 변경사항을 요약하여 리뷰어가 빠르게 이해할 수 있도록 도와줌
  
  사용 방법:
  - GitHub Actions나 웹훅을 통해 자동으로 실행
  - PR이 생성되거나 업데이트될 때마다 자동으로 분석을 수행
  - 분석 결과는 PR 댓글로 자동 게시
  
  장점:
  - 개발자의 리뷰 시간을 단축시킴
  - 일관된 코드 품질을 유지할 수 있음
  - 초보 개발자도 고품질의 코드를 작성할 수 있도록 도와줌
  - 24시간 언제든지 PR을 분석할 수 있음
