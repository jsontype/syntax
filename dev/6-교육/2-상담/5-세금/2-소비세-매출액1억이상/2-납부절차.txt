* 참조: 일본 소비세 관련 상담 0570-00-5901 건 후 5번 / 인보이스 관련 상담 03-6661-8290

1. ★ 課税仕入(카제에시이레) 분류: 25년 경비 영수증을 빼서, 매출액 20,301,800엔(税込) 중 과세대상 매입을
   다음 4구분으로 분류. 단, 소비세는 과세구분별 합계만 적는다. (소득세 경비 분류와는 별개로 다시 영수증을 봐서 나눌 것)

1. ★ 소비세 계산
  천장세금(공제 0 가정) = 税込매출 20,301,800 → 과세표준액(매출÷1.1, 천엔미만 버림) 18,456,000
    → 売上税額: 과세표준 18,456,000 × 7.8%(국세분) = 1,439,568 (지방분은 뒤에서 22/78로 별도)
    ※ 시트처럼 × 10%로 잡으면 1,845,600(국세+지방 합산 개념). 최종 결과는 같음.
  ★ [분기1] 실제 낼 소비세(일반과세) = 売上税額 − 控除対象仕入税額(과세매입 공제 합계)
  ★ [분기2] 실제 낼 소비세(간이과세) = 売上税額 × (1 − みなし매입율). IT프리랜서=第5種, みなし율 50%
            → 사실상 売上税額의 50%만 납부 (매입 입력·영수증 정리 불필요)

1. ★ 소비세 납부 (정상 신고 — 2027년분 이후 매년)
  확정신고 기간에, e-Tax에서 소득세 확정신고를 한다. 그 다음:
  >> 같은 작성코너(keisan.nta.go.jp)/e-Tax에서 이어서 「消費税及び地方消費税の確定申告書(個人事業者)」 선택
  >> 課税方式 선택: [분기1] 一般課税 / [분기2] 簡易課税 (届出 냈으면)
  >> 課税期間 입력 (예: 2027/1/1~12/31)
  >> [분기1] 一般: 4구분(課税10%·課税8%·免税10%·免税8%)별 税込매입 입력 → 세액 자동
  >> [분기2] 簡易: 매출만 입력 → みなし율 50% 자동 적용 → 세액 자동 (매입 입력·영수증 정리 불필요)
  >> 마이넘버카드로 전자서명 → 송신 → 受信通知 확인·저장
  >> 납부: 소비세(국세) + 지방소비세, 세목별 분리. 口座振替 / ダイレクト納付 / 인터넷뱅킹(ペイジー) / 신용카드 / 페이지(コンビニ) 중 택1
       ※ 인터넷뱅킹 납부는 e-Tax 송신 후 「インターネットバンキングを利用する」 → e-Gov 전자납부에서 은행(三井住友=ま行) 선택 → 로그인 → 금액 자동
  >> ★납부기한 주의: 개인사업자 소비세 = 3/31 (소득세 3/15보다 늦음)
